# Pokemon Showroom Services in NestJS

## Goals

- To provide the exact same services as [Express Pokemon Showroom Services](https://github.com/altRush/express-pokemon-showroom-services) but in NestJS way
- This NestJS service is expected to be used with [React Pokemon Showroom](https://github.com/altRush/react-pokemon-showroom.git) interchangably by said "Express" build
  - Same DB
  - Same API interfaces
  - Same expected response
  - Same exact use cases
  - Just different framework

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
